Speedia

by Data Ronin

The purpose of this extesnion is simple, to provide a quicker way to access all media on the web. Text audio and video.

Speedia is built around some simple controls and keybindings

controls/features

  • play button: Double click to read from clipboard, Click once to replay last read
  • pause button: Pauses speed reading
  • restart button: Returns speed reader to start of last read
  • arrow left: Reduces words per minute
  • arrow right: Increases words per minute
  • slider: Controls media playback speed

Hotkeys

hotkeyFunction
ctrl + X open speed media
ctrl + shft + h come back here to see hotkeys
 
ctrl + space play/pause reading from current stored data
ctrl + shift + double tap space start new reading with data from clipboard (replaces previously stored reading)
ctrl + shft + UpArrow Turn up reading speed
ctrl + shft + DownArrow Turn down reading speed
ctrl + shft + LeftArrow Turn down media playback speed
ctrl + shft + RightArrow Turn up media playback speed

Return here at any time by pressing ctrl+ shift + h
